# Group 15 — Python Quests

A collection of 30 Python exercises that walk through the fundamentals of programming, starting from the very basics and building up to small projects.

## What We Learned

Through these quests we practiced:

- **Printing output** with `print()` and formatting strings with f-strings
- **Variables and data types** — strings, integers, and floats
- **User input** with `input()` and type conversion using `int()` and `float()`
- **Arithmetic operators** — addition, subtraction, multiplication, division, integer division (`//`), and modulus (`%`)
- **Conditionals** — `if`, `elif`, `else`, nested conditions, and logical operators (`and`, `or`)
- **Loops** — `for` loops with `range()`, `while` loops, `break`, and input validation
- **Functions** — defining functions with `def`, using parameters, return values, and combining multiple functions together
- **Mini projects** — a number-guessing game, a simple calculator, FizzBuzz, and a text-adventure game

## How to Run

```bash
python quests/quest_1_first_spell.py
```

Replace the filename with any quest you'd like to try.
